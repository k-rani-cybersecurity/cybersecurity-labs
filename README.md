# Cybersecurity Labs

A collection of hands-on cybersecurity labs focused on practical learning, security testing methodologies, and defensive analysis.

This repository documents my work across various cybersecurity domains, including reconnaissance, vulnerability assessment, web application security, and network traffic analysis.

> **Disclaimer:** All activities documented in this repository were performed in authorized lab environments, intentionally vulnerable applications, or safe test systems. No testing was conducted against unauthorized targets.

## Labs Included

### 1. Reconnaissance & Vulnerability Scanning

**Objective:** Perform active reconnaissance and identify potential security risks in a controlled environment.

#### Tools Used

* Nmap
* Nikto
* OpenVAS

#### Activities

* Port and service enumeration
* Service version detection
* Web server vulnerability scanning
* Risk identification and documentation

#### Deliverables

* Scan result screenshots
* Vulnerability reports
* Security recommendations

### 2. Web Application Security Basics

**Objective:** Identify common web application vulnerabilities in a safe testing environment.

#### Tools Used

* Burp Suite
* DVWA (Damn Vulnerable Web Application)

#### Activities

* Cross-Site Scripting (XSS) testing
* SQL Injection testing
* HTTP request interception and analysis
* Mitigation planning

#### Deliverables

* Screenshots of findings
* Proof-of-concept demonstrations
* Mitigation recommendations

### 3. Wireshark Packet Analysis

**Objective:** Capture and analyze network traffic to understand protocol behavior and identify suspicious activity.

#### Tools Used

* Wireshark

#### Activities

* HTTP traffic analysis
* DNS query inspection
* TCP three-way handshake analysis
* Packet flag examination
* Identification of unusual or malformed packets

#### Deliverables

* Packet capture screenshots
* Traffic analysis reports
* Protocol observations
